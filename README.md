# Titan Eyeplus Delivery Experience

A simple static feedback form for Titan Eyeplus delivery experience.

## Project Overview

This project is a client-side web app built with:
- `index.html` — form structure and flow
- `styles.css` — layout and styling
- `script.js` — form logic, validation, and navigation

## Features

- Multi-page feedback form
- 0–10 satisfaction rating with color-coded buttons
- Required additional feedback before continuing
- Follow-up ratings for billing ease, value for money, store look and feel, and staff helpfulness
- Success message after submission

## How to Run Locally

1. Open `index.html` in your browser.
2. Alternatively, serve the folder with a local web server.

### Using Python (if installed)
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.

## GitHub Pages

To publish with GitHub Pages:
1. Push the project to GitHub.
2. In the repository, go to `Settings` → `Pages`.
3. Select branch `main` (or `1`) and folder `/ (root)`.
4. Save.

Your site should be available at:

`https://lwcx.in/?n=W1nfpDNXHtC4Gb`

## Notes

- The current version is static and does not store data on a server.
- Feedback is collected in the browser and displayed on the success page.
